Explorando HTML y CSS

HTML

El Lenguaje de Marcado de Hipertexto o HTML es un lenguaje de marcado que se utiliza para estructurar y desplegar una página web y sus contenidos. Por ejemplo, sus contenidos podrían ser párrafos, una lista con viñetas, o imágenes y tablas de datos.

CSS

CSS o Hojas de Estilo en Cascada es el código que se usa para dar estilo a una página web, este no es realmente un lenguaje de programación, ni tampoco es un lenguaje de marcado, sino un lenguaje de hojas de estilo que permite aplicar estilos de manera selectiva a elementos en documentos HTML.
